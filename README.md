# velair-ui

## Установка в проект

Из npm (после публикации пакета):

```bash
npm install velair-ui lit
```

`lit` указан как peer-зависимость в `velair-ui`: в приложении должна быть установлена совместимая версия **Lit ^3**.

Если пакета ещё нет в реестре, можно поставить из репозитория или с диска:

```bash
npm install github:<user>/velair-ui lit
```

```bash
npm install file:../путь/к/velair-ui lit
```

В обоих случаях по-прежнему добавьте `lit` в зависимости проекта.
